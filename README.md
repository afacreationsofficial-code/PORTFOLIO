📁 Portfolio Manager
✨ Interactive Personal Portfolio & Document Organizer

A modern, pastel-themed portfolio management web application that helps users organize documents, achievements, projects, and certificates in one beautiful, structured workspace — and generate a professional, shareable portfolio effortlessly.

Think Notion × Google Drive × Personal Website Builder — built specifically for career growth.
